# HHYDB
This is a simple database capstone project of Database Introduction course of the Department of Computer Science and Technology, Tsinghua University.

## Dependencies

* g++
* cMake version >= 3.16
* bison and flex

## Environment

Tested on WSL2 Ubuntu-20.04.

## Build

```bash
cd HHYDB
make
```

## Execute

```bash
./HHYDB
# sql commands examples
drop database db1;
create database db1;
use db1;
```
